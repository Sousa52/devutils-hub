# DevUtils Hub ⚡

A clean, modern collection of useful developer tools and utilities — all in one place.

Built as a lightweight, single-file dashboard for quick access to everyday dev tools.

![Preview]("https://github.com/user-attachments/assets/c2e48a8d-fe38-4085-9f0a-5000f8154012")  

## ✨ Features

- **JSON Formatter & Validator**
- **Base64 Encoder / Decoder**
- **URL Encoder / Decoder**
- **JWT Decoder**
- **Color Picker & Converter** (Hex, RGB, HSL)
- **Lorem Ipsum Generator**
- **Password Generator**
- **Markdown Preview**
- **Regex Tester**
- **Timestamp Converter**
- Dark mode + responsive design
